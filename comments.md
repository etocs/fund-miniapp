## 🆕 新增需求：截图导入基金功能

### 功能说明
添加通过截图识别导入基金信息的功能，使用**微信官方 OCR 插件**（完全免费）。

### 具体需求

#### 1. 新增页面：`pages/import/` - 截图导入页
- 支持拍照或从相册选择图片
- 使用微信官方 OCR 识别图片中的文字
- 智能提取基金代码（6位数字）
- 识别持仓信息（份额、成本价等）
- 显示识别结果供用户确认
- 支持手动修改识别结果
- 批量添加到自选基金

#### 2. 首页添加入口
- 在首页添加"截图导入"按钮
- 点击跳转到导入页面

#### 3. OCR 识别功能实现

**配置 app.json 添加插件：**
```json
"plugins": {
  "ocr-plugin": {
    "version": "1.3.0",
    "provider": "wx4418e3e031e551be"
  }
}
```

**实现识别逻辑：**
```javascript
// 调用微信官方 OCR
const ocrPlugin = requirePlugin("ocr-plugin")
const ocrInstance = ocrPlugin.OcrNavigator

// 识别图片
wx.chooseImage({
  success: (res) => {
    const tempFilePath = res.tempFilePaths[0]
    // 使用 OCR 识别
    ocrInstance.recognize({
      path: tempFilePath,
      success: (res) => {
        // 解析识别结果
        const text = res.result
        // 提取基金代码和信息
      }
    })
  }
})
```

#### 4. 智能解析算法

**识别规则：**
- 基金代码：6位连续数字（如：001234、519989）
- 持有份额：匹配"份额"、"持有"关键字附近的数字
- 成本价：匹配"成本"、"成本价"附近的数字
- 市值：匹配"市值"、"金额"附近的数字

**正则表达式示例：**
```javascript
// 匹配基金代码
const fundCodeReg = /\b\d{6}\b/g

// 匹配份额
const shareReg = /(?:份额|持有)[：:]\s*(\d+\.?\d*)/  

// 匹配成本价
const costReg = /(?:成本价|成本)[：:]\s*(\d+\.?\d*)/  
```

#### 5. 用户交互流程

1. 用户点击"截图导入"
2. 选择拍照或从相册选择
3. 显示加载状态"识别中..."
4. 展示识别结果列表：
   - 基金代码
   - 识别到的份额
   - 识别到的成本价
   - 每项支持手动编辑
5. 用户确认或修改
6. 点击"导入"按钮
7. 自动添加到自选基金
8. 如果识别到份额和成本价，同时保存到持仓计算器
9. 跳转回首页并刷新
